About Remove Metadata

PDF metadata can contain identifying information. This tool creates a new PDF that copies visible pages but does not copy the original document's metadata, providing a privacy-focused result. Everything runs in your browser so no data leaves your device.

Step-by-step guide

  1. Upload a PDF.
  2. Click "Remove Metadata" to generate a stripped copy.
  3. Download the resulting file.

FAQ

What metadata is removed?

Common fields like author, title, and creation dates are not copied when creating the new document.

Is content changed?

Visible content is preserved; only metadata is removed.
